서버실행하는 거에 있어서 로컬로 밖에 안되기 때문에  
프로젝트 다운 받으시면  
vscode등을 통해서 실행 하시면  
main.py의 fastapi 서버를 실행해서  
터미널화면 : uvicorn main:app --reload 코드 실행 후  
index.html을 여시면 진행할 수 있습니다.
